# Project Overview: CS4223 Spring 2026 Weather App Challenge

## Context
This project is an extra-credit assignment for the CS4223 Software Quality and Testing course. The goal is to extend two existing Flask-based weather applications with new features while maintaining software quality.

## Objectives
1. **Understand existing codebases:** Analyze `2025f-feature-development-b` and `2025f-feature-development-c`.
2. **Implement Dynamic Background Themes:** Background changes based on weather/time.
3. **Implement 5-Day Forecast Extension:** Display daily forecasts below current weather.
4. **Implement Weather-Based Suggestions:** Display dynamic advice based on weather.
5. **Quality and Testing:** Ensure functional correctness and adhere to coding standards.

## Target Repositories
- **Repository B:** `2025f-feature-development-b/`
- **Repository C:** `2025f-feature-development-c/`

## Technology Stack
- **Repository C (Desktop App):**
  - **Backend/UI:** Python (Tkinter)
- **Repository B (Web App):**
  - **Backend:** Python (Flask) as a JSON API
  - **Frontend:** React with Tailwind CSS
- **APIs:** OpenWeather API
